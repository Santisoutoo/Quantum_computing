## Ejecución

### Instalar dependencias

```bash
pip install -r requirements.txt
```

### Ejecutar todo

```bash
python main.py
```

### Ejecutar entregables individuales

```bash
# Multithreading
python multithreading/ejemplo_uso.py

# Control clásico
python control_clasico/ejemplo_uso.py
```
